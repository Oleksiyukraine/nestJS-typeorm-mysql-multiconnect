[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Description
```bash
    Main goal is to get data from separate DB according to route

    In this template was implemented:
    1)Database multiconnect
    2)Architecture Controller->Service->Repository
    3)Base repository
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
